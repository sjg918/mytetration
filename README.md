# mytetration
DMTPark 복소수 테트레이션 연산 수렴/발산 집합 시각화</br>
주요구현: 다중GPU를사용한병렬처리, 자동edge추적을통한영상딸깍생성</br>
사용된library: Pytorch</br>
1개의 영상을 처리하는데 필요한 시간</br>
RTX3090 1대: 0.765s</br>
RTX3090 4대: 0.238s</br>
